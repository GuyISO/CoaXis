# Viewerに実装予定の機能

## 選択処理関係
矩形での複数選択
    ドラッグで矩形を描いて範囲指定まではOK、AABBで大まかに選択してから、さらにColliderで判定
    完全に含むか、一部でも含むかの両方を選択できるようにする
Ctrlを押しながらのモデル複数選択
    モード切替で、Ctrlを押しながらの選択を有効にする
Treeの複数選択
    Treeが標準で持つMultiSelectを有効にし、対応させる
配列を返す貫通レイキャスト
    現状は最初にヒットしたオブジェクトのみ選択される仕様になっているが、レイキャストの貫通を実装して複数オブジェクトから選択できるようにする

## カメラ・表示関係
複数ノードのフィット機能
    Fit機能は単一のNode3DもしくはNodeを配列(IEnumerable)で受け取るオーバーロードを持つこととする
CullMaskの切り替え
    カメラのCullMaskを切り替えることで、特定のレイヤーに属するオブジェクトの表示/非表示を制御する機能を実装する

## 3D表示関係
選択状態のモデルの色変更
    Materialで制御？
    色やMaterial以外で3Dモデルを目立たせる方法を用意、発光点滅など
点群座標へのVFX配置
    GodotのVFX Graphを使用して、点群座標にエフェクトを配置する機能を実装する
3Dから2Dに投影してテキストを表示
    3D空間の特定の位置にテキストを表示するために、カメラ投影を使用して2Dスクリーン座標に変換し、UI上にテキストを配置する機能を実装する
3D空間上へのテキスト表示
    3D空間上にテキストを表示するための機能を実装する。これには、3Dテキストオブジェクトを作成し、特定の位置に配置する方法が含まれる
選択ノードの表示/非表示切替
    Godotの表示非表示切り替えで実装する

## IPC
IPC通信の実装
    GodotのTCP/IP通信機能を使用して、外部プロセスとの通信を実装する
    JSONやProtobufなどのデータフォーマットを使用して、効率的なデータ交換を行う

## UI
オーバーレイのような透過する仕様ツリー
    Godotの仕様上むり？検討する
Window管理
    各UIをWindowの入れ子にする
Dockable Window
    各UIをDock可能なContainerの入れ子にして使えるようにもする
設定画面
    ビューワーの各種設定を行うためのUIを実装する

## ログ
ログの実装
    ビューワーの動作やエラーなどの情報を記録するためのログ機能を実装する。これには、ログレベルの設定やログファイルへの出力などが含まれる

## 設定
設定画面とsettings.jsonの連携
    設定画面で変更された内容をsettings.jsonに保存し、ビューワーの動作に反映させる機能を実装する

## 計算
CATIAが持つ右手系Z-up座標系への対応
    Godotは左手系Y-up座標系を使用しているため、CATIAの右手系Z-up座標系に対応するための変換処理を実装する
