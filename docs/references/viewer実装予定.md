# Viewerに実装予定の機能

## 細かい修正
測定機能
    表示されるラベルと線を豪華にする、
    配置する親ノードを見直して選択されないようにする、（UIの子で入れてしまえる？）
PickHandleモードのUI反映
    現在の状態が各UIでわかるように表示(トグルボタン化など)
    大元のPickHandleモードをPickEventHubに持たせるようにする？
選択
    複数選択でもMultiSelectだとToggleになっているので使いづらい、AddかRemoveかのどちらかにする

## 3D表示関係
点群座標への点モデル配置
    VectorからQuaternionに変換どうする？とりあえず直変換で
点群座標をカメラ投影しての点配置
    とりあえず3Dモデルで始めるのでまだ先
モデルのマテリアル変更以外のハイライト方法
    色やMaterial以外で3Dモデルを目立たせる方法を用意、発光点滅など
点群座標へのVFX配置
    GodotのVFX Graphを使用して、点群座標にエフェクトを配置する機能を実装する
3Dから2Dに投影してテキストを表示
    3D空間の特定の位置にテキストを表示するために、カメラ投影を使用して2Dスクリーン座標に変換し、UI上にテキストを配置する機能を実装する
3D空間上へのテキスト表示
    3D空間上にテキストを表示するための機能を実装する。これには、3Dテキストオブジェクトを作成し、特定の位置に配置する方法が含まれる

## UI
オーバーレイのような透過する仕様ツリー
    Godotの仕様上むり？検討する
Dockable Window
    各UIをDock可能なContainerの入れ子にして使えるようにもする
設定画面
    ビューワーの各種設定を行うためのUIを実装する
    Settings.jsonの内容を読み込み、設定画面に反映させる機能を実装する
    設定変更したら、Settings.jsonに保存する機能を実装する
Undo/Redo
    履歴表示UI
貫通レイキャスト
    一覧と選択UI

## 設定
設定画面とsettings.jsonの連携
    設定画面で変更された内容をsettings.jsonに保存し、ビューワーの動作に反映させる機能を実装する

## ユーティリティ
アセットマネージャー
    個々のクラス内でPackedSceneやアセットのパス指定するのやめたい
設定変更EventHub追加
    設定変更時に、他のコンポーネントに通知するためのEventHubを追加する。これにより、設定変更がビューワー全体に反映されるようになる
Modelの移動機能
    3D空間上でModelを移動させる機能を実装する。これには、ユーザーがModelを選択し、ドラッグ操作で移動させる機能が含まれる？
    Snapping機能
    Modelの移動時に、特定のグリッドやポイントにスナップさせる機能を実装する。これにより、Modelの配置が正確に行えるようになる
